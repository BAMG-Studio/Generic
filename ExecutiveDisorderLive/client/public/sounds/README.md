# Sound Effects for Executive Disorder

## Available Sounds

- **success.mp3** - Positive decision outcome
- **disaster.mp3** - Negative decision outcome  
- **neutral.mp3** - Neutral decision outcome
- **alert.mp3** - Crisis or cascade effect warning
- **dramatic.mp3** - Ending sequence music
- **click.mp3** - UI interaction sound
- **theme.mp3** - Main menu theme music
- **ambient.mp3** - Background ambient sound

## Usage

Sounds are loaded dynamically via the game's audio system. They are played contextually based on:
- Decision outcomes (positive/negative resource changes)
- Crisis events
- Cascade effects
- Game endings
- UI interactions

## Format

All sounds are in MP3 format, optimized for web playback.
