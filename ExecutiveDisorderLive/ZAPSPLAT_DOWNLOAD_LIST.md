## üí° Pro Tips

**File Format:**
- Download as **MP3** (best compatibility)
- Fallback: OGG Vorbis
- Avoid WAV (too large for web)

**File Size:**
- UI sounds: Keep under 50KB each
- Effects: 50-200KB
- Ambient loops: 500KB-2MB max
- Music stings: 200-500KB

**Naming:**
- Use lowercase
- Separate words with hyphens
- Be descriptive: `card-flip.mp3` not `sound1.mp3`

**Search Tips on Zapsplat:**
- Use quotes for exact phrases: "card flip"
- Try multiple search terms if first doesn't work
- Preview sounds before downloading
- Check the "Free" filter to avoid premium sounds
